html : 홈페이지 *구조* 를 담당 / 문서의 틀을 만드는 작업을 한다.
css : 스타일 을 담당 / 
java script :  * 상호작용 을 담당  / html 이나 css 로 작업한 내용을 작동하게 해준다 .

ui : 사용자가 보는 인터페이스를 말한다. 
ux : 사용자의 느낌 태도 행동을 말한다. ( 사용자 중심의 개성적인 디자인? 평가? 을 말하는듯 )


----------
html : ( <> ) 부등호로 감싸진 tag 들의 집합이다 .
트리구조로 이루어져 있다. ( 구글검색 : 트리구조 ) 
태그에는 style , src 등 속성을 담을 수 있다
자주 사용하는 요소 ( 속성 )  
div : 한 줄을 사용한다는 태그 
span : 컨텐츠 크기만큼 공간을 사용하는 태그 
img : 이미지를 삽입 할 수 있는 태그 
a : 링크를 삽입 해 줄수 있는 태그 ( href 속성을 사용해야 한다.) 
ul / li : 리스트를 만들 수 있는 태그 / ul (리스트를 만들 수 있다고 선언) 안에 li (list)를 작성 할 수 있다.
input : 무언가를 입력할 수 있는 공간을 만들 수 있다. 
textarea : 택스트를 적을 수 있는 공간을 만들어준다. 
button :  누를 수 있는 버튼을 만들어준다.

css 
셀렉터 { } 로 구성되어 있다
셀렉터는 body , title 같은 특정 태그 이름을  지정하는것이고
{ }  : 선언 블록에서는 color , font-size 같은 요소를 집어 넣을 수 있다. 

id 는 문선 내에 단 하나의 요소에만 적용 할 수 있다. 
<li id="menu-item">Home</li>
특정한 요소에 이름을 붙이는데 사용한다.

같은 이름을 사용하고 싶다면 id 대신 class 를 사용하면 된다 . 
동일한 값을 갖는 여러 요소를 사용 할 수 있다.
스타일 분류에 사용한다.
<li class="menu-item">Home</li>
<li class="menu-item">Mac</li>

border ( 테두리 )
margin ( 바깥 여백 )
padding ( 안쪽 여백 )
